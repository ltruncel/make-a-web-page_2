1.1 The Basics of the Web and HTML

World Wide Web

The web is collection of HTML documents. HTML stands for Hypertext Markup Language. Examples of files can be found on the web include plain text, HTML, videos, images and music. When a user goes to a website, her computer sends a HTTP request to a server and the server finds the corresponding HTML documents and sends it back to her computer where a web broswer interprets the pages and displays them.

HTML Tags

Tags are composed of an opening tag, the contents, and a closing tag. Tags represent various elements of HTML, through which we make commands to the browsers.

1.2 Creating a Structured Document with HTML

Google Developer Tools

I can use Google developer tools to see the code of a webpage.

Tree-Like Structure

This refers to the fact that HTML elements can have other elements embedded in them. In a browser this structure is manifested as nested boxes.

HTML and CSS

HTML and CSS are both "languages." HTML controls the "structure" of a web page. CSS controls how a page "looks."

Document Object Model

When programmers talk about the "DOM" (Document Object Model), they are talking about the tree-like structure of a page.

1.3 Using CSS to Add Style to HTML Structure

Cascading Style Sheets

CSS stands for Cascading Style Sheets. It refers to the way programmers control the style of HTML elements. By giving similar elements the same classname programmers can then specify the style to be applied to that class.

The Box Model

The box model includes the margin, border, padding and content. By default, the size of the actual element is the sum of the border, padding and content width.

Avoiding Repetition

Avoiding repetition is important because doing so allows programmers to avoid errors that may arise when they copy and paste the same code over and over.
